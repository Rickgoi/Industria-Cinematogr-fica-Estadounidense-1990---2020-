# Análisis Exploratorio de la Industria Cinematográfica Estadounidense (1990-2020)

## Descripción del Proyecto
Análisis completo de la industria cinematográfica estadounidense utilizando datos de Box Office Mojo e IMDB para identificar tendencias, patrones y factores de éxito.

##  Objetivos
- Identificar géneros y clasificaciones más rentables
- Analizar tendencias temporales en presupuestos y recaudaciones
- Determinar factores clave del éxito cinematográfico

## Tecnologías Utilizadas
- Python 3.8+
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- Scikit-learn
- Jupyter Notebook
